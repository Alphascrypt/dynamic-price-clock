TLS-Zertifikatspruefung Tibber-API

Geprueft

Root-CA-Zertifikat hinterlegt. TLS-Verbindungen zur Tibber-API werden geprueft.

So ermittelst du das aktuelle Zertifikat (auf einem PC, NICHT auf dem ESP32):
openssl s_client -connect api.tibber.com:443 -showcerts </dev/null | openssl x509 -outform PEM
Das letzte Zertifikat der ausgegebenen Kette (Root-CA) hier komplett inkl. BEGIN/END-Zeilen einfuegen.
